# 🪑 WoWo Furniture Backstage — 後台管理系統

👉 專案連結： [前台網站](https://wayne0917.github.io/Wayne0917-project-wowoFurniture.io/)
 ｜ [後台管理系統](https://wayne0917.github.io/Wayne0917-project-wowoFurnitureBackstage.io/)

🧭 專案簡介

此專案為 WoWo Furniture 家具電商平台的後台管理系統，
提供管理者對訂單進行檢視、狀態更新、刪除與資料統計的功能，
以模組化程式架構設計，方便維護與擴充。

⚙️ 功能介紹
📦 訂單管理

查看所有訂單資料（姓名、電話、地址、付款狀態、金額等）

✅ 切換訂單處理狀態（已處理／未處理）

🗑️ 支援刪除單筆與全部訂單

🔍 可依付款狀態篩選訂單

📊 資料分析

使用 C3.js 製作圓餅圖顯示：

各品項銷售比例

各類別銷售比例

即時更新圖表，方便觀察銷售趨勢

🧩 使用技術
類別	技術
前端語言	HTML5、CSS3、JavaScript (ES6+)
API 串接	Axios
圖表套件	C3.js、D3.js
架構概念	MVC 模式

🚀 快速啟動

下載專案：

```bash
git clone https://github.com/Wayne0917/Wayne0917-project-wowoFurnitureBackstage.io.git
```

開啟 index.html 即可在瀏覽器中查看後台畫面。
